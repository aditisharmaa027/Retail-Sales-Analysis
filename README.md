# Retail-Sales-Analysis

Business Statement - A retail Store would like to understand customer behaviour using their point of sales data (POS).A retail store wants to understand customer behavior, sales performance, and product trends using their point-of-sale (POS) data. The goal is to optimize inventory, improve sales strategies, and reduce returns across store types and product categories.

Overview - This project analyzed retail sales, customer, and product data to uncover transaction trends, revenue patterns, and return behavior across different product categories and store types.


Key Actions / Work Done - 
Queried a relational database using SQL, performing joins, aggregations, and time-based analyses
Calculated key metrics such as total revenue, top-selling products, and store performance
Analyzed returns and identified product categories and customer segments with high return rates
Conducted sales trend analysis over time and across store types


Insights / Impact - 
Identified high-performing products and categories driving most revenue
Found store types with the highest sales
Highlighted customer segments and products with high returns
Insights can help optimize inventory, improve sales strategy, and reduce losses due to returns
Certain channels are preferred by most customers, indicating opportunities for targeted campaigns and operational focus.


Tools & Technologies - 
SQL (Joins, Aggregations, Date Functions, Subqueries)
Relational Databases (POS Data)
