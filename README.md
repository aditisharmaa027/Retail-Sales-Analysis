# Retail-Sales-Analysis

Business Statement - A retail Store would like to understand customer behaviour using their point of sales data (POS).

Overview - "This project was about analyzing retail sales, customer, and product data to understand tranaction trends, sales performance and returs across different product categories and store types."
What I did (Actions Taken) - "Queried a relational database using SQL performing joins, aggregations and time-based analysis.
Insights - " From this analysis, I identified high-performing product, categories, stored with most sales, and segments with high returns. These insights could help decision makers optimise inventory, improve sales strategy, and reduce returns.
